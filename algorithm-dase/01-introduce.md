# 绪论

## 常见分布的期望和方差

* 0-1分布 (伯努利分布): $E = p$, $Var = p(1-p)$
* 二项分布 (n个独立的伯努利试验中成功的次数的离散概率分布): $E = np$, $Var = np(1-p)$
* 泊松分布: $E = \lambda$, $Var = \lambda$
* 正态分布: $E = \mu$, $Var = \sigma^2$
* 均匀分布: $E = \frac{a+b}{2}$, $Var = \frac{(b-a)^2}{12}$

## 方差

对于独立随机变量$X_1$, $X_2$. $Var(X_1 + X_2) = Var(X_1) + Var(X_2)$
